📌 Overview
CodeAlpha LT2 is a fast and intuitive web-based translation tool. It leverages powerful APIs to provide real-time text conversion, helping users break down language barriers with a single click.

🚀 Key Features
Multi-Language Support: Instant translation across dozens of global languages.

Auto-Detect: Smart detection of the source language for faster input.

User-Friendly UI: A clean, focused design for seamless desktop and mobile use.

Copy-to-Clipboard: Quick one-click copying of translated text.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript

API: Translation API Integration

Deployment: GitHub Pages
